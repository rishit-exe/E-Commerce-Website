## Overview

Furniro is a fully responsive e-commerce furniture website frontend built with Next.js and TypeScript. The design was created in Figma and then implemented with pixel-perfect accuracy.

## Tech Stack

- **Next.js** 16.0.7
- **React** 19.0.0
- **TypeScript**
- **Tailwind CSS**
- **Turbopack**

## Setup

```bash
# Install dependencies
npm install

# Build for production
npm run build

# Start production server
npm run start
```

## Development

```bash
# Start development server
npm run dev
```
